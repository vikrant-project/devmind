# Usage
- `devmind version` / `devmind config` / `devmind hardware` / `devmind models`
- `devmind build "<prompt>"` — main command
- `devmind build "<prompt>" --dry-run` — show plan only
- `devmind continue <workspace>` — resume an interrupted build
- `devmind list` / `devmind inspect <ws>` / `devmind export <ws>`
- `devmind benchmark --suite mvp` / `devmind pull <model>` / `devmind clean <ws>`
